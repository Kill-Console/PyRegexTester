RegexTester
===========

A PyQt regex tester

A simple regex tester by PyQt, it can test regular expressions. If the regex matches some data, the matched data will be highlighted. It supports three kinds of modes:

 1. IGNORECASE (re.I)
 2. MULTILINE (re.M)
 3. DOTALL (re.S)
